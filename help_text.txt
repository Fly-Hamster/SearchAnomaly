Эксплуатация:
1. Необходимо выбрать файл с форматом *.csv. Должно быть минимум 2 столбца: первый - временной показатель, второй - значения временного ряда.
2. Далее необходимо ввести номер столбца, в котором будет производиться поиск. Если не выбран - поиск по умолчанию будет осуществлен в первом.
3. Нажать кнопку "Начать поиск". Будет проанализирован файл, определена основная информация о ряде и запущен поиск аномалий.
4. Результат можно сохранить только после полного анализа документа. Сохранение в формате *.csv.
Основные понятия:
1. Максимальное, минимальное, среднее значение - значения, определенные в ряду. Показаны для того, чтобы в дальнейшем проверить являются ли они аномалией.
2. Количество строк - длина ряда.
3. Общее количество аномалий - суммарное число всех видов аномалий.
